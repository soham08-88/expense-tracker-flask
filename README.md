# Expense Tracker (Flask)

A simple Expense Tracker web application built using Flask.  
Users can add, edit, delete, and analyze their expenses with charts.

## 🚀 Features
- User Login & Register
- Add / Edit / Delete Expenses
- Dashboard with category-wise chart
- CSV export functionality
- Clean UI with sidebar
- User-specific data storage

## 🛠️ Tech Stack
- Python (Flask)
- SQLite Database
- HTML, CSS
- Chart.js
- Pandas

## ▶️ How to Run Locally
1. Clone repository
2. Install dependencies

pip install -r requirements.txt

3. Run application

python app.py

4. Open browser

http://127.0.0.1:5000/
